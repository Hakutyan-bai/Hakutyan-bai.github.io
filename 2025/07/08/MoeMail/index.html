MoeMail临时邮箱

📮 MoeMail：基于 Mail.tm 的轻量级临时邮箱平台

MoeMail 是一个现代化、界面优雅的临时邮箱系统，基于 Mail.tm API 构建，并集成 Vue 前端和 Golang 后端，部署方便、使用顺畅，非常适合需要匿名收信或临时注册的场景。

✨ 项目特点

  • 🌐 临时邮箱支持：基于 mail.tm 提供的免费临时邮箱 API
  • 🖥️ 现代前端：使用 Vue 3 + Tailwind CSS 构建，简洁优雅，响应式设计
  • ⚡ 高效后端：Golang 编写，快速、可靠、轻量
  • 🚀 开箱即用：提供 Docker Compose 支持，几行命令即可部署运行
  • 🔒 数据不持久化：匿名接收邮件，即用即焚

🖼️ 界面展示

前端界面极其简洁清爽，主要功能包括：

  • 随机邮箱地址生成
  • 邮件列表显示
  • HTML 和纯文本内容预览
  • 一键复制邮箱 / 验证码

🛠️ 适用场景

MoeMail 特别适合以下场景使用：

  • 注册网站时避免暴露真实邮箱
  • 接收一次性验证码或验证邮件
  • 防止垃圾邮件干扰主邮箱
  • 测试 Web 项目邮箱通知功能

💬 总结评价

MoeMail 是一个结合了实用性与美观性的临时邮箱工具，部署简单，界面轻快，适合个人用户和开发者测试使用。如果你正在寻找一款可控、可自建的临时邮箱系统，它会是一个非常好的选择！
